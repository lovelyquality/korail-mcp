# 🏠 m-internal-svc — 사내 서비스 조회

KORAIL 임대·사회공헌·구내식당·회의실·직급 등 사내 서비스를 조회하는 MCP 서버 (도구 14개)

| 도구 | 설명 |
|---|---|
| `get_lease_stores` | 임대 점포 조회 |
| `get_lease_codes` | 임대 코드 조회 |
| `get_leased_assets` | 임대 자산 조회 |
| `get_dormitory_longterm_codes` | 기숙사 장기 코드 조회 |
| `get_social_funds` | 사회공헌 기금 조회 |
| `get_social_volunteer_fields` | 봉사활동 분야 조회 |
| `get_social_donations` | 기부 정보 조회 |
| `get_social_volunteer_matching` | 봉사 매칭 정보 |
| `get_social_org` | 사회공헌 조직 조회 |
| `get_support_facilities` | 복지 시설 조회 |
| `get_support_departments` | 지원 부서 조회 |
| `get_office_meeting_rooms` | 회의실 조회 |
| `get_job_grades` | 직급 정보 조회 |
| `get_cafeteria_menu_stats` | 구내식당 메뉴 조회 |
